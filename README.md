# Audi AG — Financial Modeling & Valuation (FP&A)

## Project Overview
This project is a comprehensive **financial planning and analysis (FP&A) model** built for Audi AG using publicly available annual report data.

The model is designed to replicate real-world corporate finance and FP&A workflows, including forecasting, scenario analysis, and valuation.

---

## Objectives
- Analyze historical financial performance (2022–2024)
- Build driver-based forecasts (2025–2027)
- Link Income Statement, Balance Sheet, and Cash Flow Statement
- Estimate enterprise value using Discounted Cash Flow (DCF)
- Support strategic decision-making through scenario analysis

---

## Model Structure
The Excel model includes:

- **Drivers sheet**
  - Revenue growth
  - Cost and margin drivers
  - Working capital assumptions
  - CapEx and depreciation
- **3-Statement financial model**
  - Income Statement
  - Balance Sheet
  - Cash Flow Statement
- **Forecast period:** 2025–2027
- **Scenario analysis:** Base / Bull / Bear
- **Valuation**
  - Free Cash Flow to Firm (FCFF)
  - WACC-based DCF
  - Sensitivity analysis
- **Executive dashboard**
  - Key financial KPIs
  - Scenario comparison
